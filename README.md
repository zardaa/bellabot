## CARA INSTALL
# TERMUX
```bash
> download termux
> buka
> pkg install git
> pkg install ffmpeg
> pkg install nodejs
> apt update && apt upgrade
> git clone https://github.com/benisadewa/reuzsc
> cd reuzsc
> bash install.sh
> node index.js
```
